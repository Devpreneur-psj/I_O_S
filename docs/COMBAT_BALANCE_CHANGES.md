# 전투 밸런스 조정 - 초보자 친화성 개선

## 변경 날짜
2025-11-29

## 개요
전투 & 모험 컨텐츠에서 초보자도 접근 가능하도록 밸런스를 조정했습니다.
정령이 기술을 배우지 않았거나 모든 기술이 쿨타임 중일 때도 전투에 참여할 수 있도록 개선했습니다.

## 주요 변경 사항

### 1. 기술 없을 때도 기본 공격 가능
**문제**: 기술이 없거나 모든 기술이 쿨타임 중이면 공격을 스킵하여 패배하는 문제

**해결**:
- 기술이 없거나 모든 기술이 쿨타임 중일 때 **기본 공격**을 자동으로 사용
- 기본 공격 데미지에 1.2배 보너스 적용

**파일**: `CombatService.java`
- 라인 222-291: 기술이 없을 때도 공격을 수행하도록 수정

### 2. 기술 데미지 효과 강화
**변경 전**:
- 기술 위력 = basePower * (1.0 + (masteryLevel - 1) * 0.125)
- 숙련도 1레벨: 100%, 5레벨: 150%

**변경 후**:
- 기술 위력 = basePower * (1.1 + (masteryLevel - 1) * 0.175) * 1.3
- 숙련도 1레벨: 143% (110% * 1.3), 5레벨: 234% (180% * 1.3)
- **기술 사용 시 30% 추가 데미지 보너스** 적용

**파일**: `CombatService.java`
- 라인 404-419: 기술 데미지 계산 로직 개선

### 3. 적 능력치 낮추기
**변경 전**:
- 적 기본 능력치 = 레벨 * 2
- 보스 능력치 = 레벨 * 2 * 1.5 = 레벨 * 3
- 적 HP = (레벨 * 10) + 50
- 보스 HP = ((레벨 * 10) + 50) * 2

**변경 후**:
- 적 기본 능력치 = 레벨 * 1.5 (25% 감소)
- 보스 능력치 = 레벨 * 1.5 * 1.3 = 레벨 * 1.95 (35% 감소)
- 적 HP = (레벨 * 8) + 40 (20% 감소)
- 보스 HP = ((레벨 * 8) + 40) * 1.5 (25% 감소)

**파일**: `DungeonService.java`
- 라인 140-144: 적 능력치 계산 로직 수정
- 라인 196-199: 적 HP 계산 로직 수정

### 4. 플레이어 공격 데미지 보너스
**추가 사항**:
- 플레이어 공격 시 **20% 추가 데미지 보너스** 적용
- 최소 데미지: 레벨 * 2 (기존 레벨 / 3에서 대폭 증가)

**파일**: `CombatService.java`
- 라인 450-458: 최소 데미지 및 플레이어 보너스 적용

## 밸런스 개선 효과

### 초보자 경험 개선
1. **기술 없어도 전투 가능**: 초보자가 기술을 배우지 않아도 기본 공격으로 전투 참여 가능
2. **승리 확률 향상**: 적 능력치 감소 + 플레이어 데미지 증가로 승리 확률 대폭 향상
3. **기술 효과 강화**: 기술을 배웠을 때 더욱 강력한 효과를 체감

### 예상 승률 변화
- **기술 없이 전투**: 기존 0% 승리 → 변경 후 약 60-70% 승리 가능
- **기술 있이 전투**: 기존 약 50% 승리 → 변경 후 약 80-90% 승리 가능
- **보스전**: 기존 약 30% 승리 → 변경 후 약 60-70% 승리 가능

## 테스트 권장 사항
1. 기술이 없는 정령으로 전투 시도
2. 기술이 있는 정령으로 전투 시도 (기술 데미지 확인)
3. 다양한 스테이지에서 승리 확률 확인
4. 보스전 난이도 확인

## 추가 개선 제안
- 필요시 적 능력치를 더 낮출 수 있음 (레벨 * 1.3)
- 기술 숙련도 보정을 더 높일 수 있음 (최대 200% 이상)
- 특정 스테이지에서만 보너스 적용 (예: 스테이지 1-3)
